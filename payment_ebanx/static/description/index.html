Judopay Credentials

Login in Judopay.

JudoId : Settings > Acoount summary >Judo ID(s)

Secret & Token : Your Apps > (App name) >Sandbox tokens

Select your apps & go to 'Web payments configuration'

- Check the 'Enable Web payments (hosted redirect web payment)'

- Success URL :- base_url + /payment/judopay/return

- Failure URL :- base_url + /payment/judopay/return

Add Judo Id, Token & Secret. You can get Judo Id, Token and Secret from Judopay account.

Make payment using Judopay Payment Gateway, Select Judopay from Given List.It will take you to Judopay for transaction.

Judopay page for transaction.

If user card details are correct then transaction state will go in Done state and it will display confirmation message to user.

If user cancels the transaction then transaction state will go in Cancel state and it will display cancel mesaage.
